Sales & Demand Forecasting for Businesses

Project Overview

This project predicts future sales using historical business sales data. The model analyzes previous sales patterns and generates forecasts to support business decision-making.

Objective

Build a machine learning model that forecasts future sales or demand using historical sales records.

Features

- Data cleaning and preprocessing
- Time-based feature engineering
- Sales aggregation
- Forecasting using Linear Regression
- Model evaluation using:
  - MAE (Mean Absolute Error)
  - RMSE (Root Mean Squared Error)
  - R² Score
- Visualization of actual vs forecasted sales

Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

Installation

Clone the repository:

git clone <your-github-repository-link>

Install dependencies:

pip install -r requirements.txt

Run the Project

python sales_forecasting.py

Dataset

Use the provided sales dataset ("sales_dataset.xlsx") and place it in the project directory.
